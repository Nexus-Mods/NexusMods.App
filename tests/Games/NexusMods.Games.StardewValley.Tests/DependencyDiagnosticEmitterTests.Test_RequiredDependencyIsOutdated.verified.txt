[Id] NexusMods.Games.StardewValley: 2
[Title] Outdated Dependency
[Summary] 'Farm Type Manager v1.24.0' requires an updated version of 'Content Patcher 1.30.4'
[Details]
The mod **Farm Type Manager v1.24.0** requires **Content Patcher 1.30.4** version 2.0.0 or higher to function, but an older version of **Content Patcher 1.30.4** (1.30.4) is installed.

### How to Resolve
1. Download the latest version of **Content Patcher 1.30.4** (version 2.0.0 or newer) from [Nexus Mods](https://www.nexusmods.com/stardewvalley/mods/1915?mtm_source=nexusmodsapp)
2. Add the latest version of **Content Patcher 1.30.4** to the loadout
3. Remove version 1.30.4 of **Content Patcher 1.30.4** from the loadout

### Technical Details
The `manifest.json` file included with **Farm Type Manager v1.24.0** lists **Content Patcher 1.30.4** as a requirement with a minimum version of 2.0.0.
